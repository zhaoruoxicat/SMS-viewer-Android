# MySMS

一个用于查看服务器端短信记录的 Android 客户端。

本项目不直接读取手机本地短信，而是通过自定义后端接口拉取短信列表，配合(https://github.com/zhaoruoxicat/SMS-Forwarder) 使用。
应用支持首次配置、手动刷新、分页查看、验证码一键复制，以及基于系统生物识别或锁屏凭据的解锁保护。

## 功能特性

- 通过服务器地址 + Token接入短信后端
- 首次打开自动进入配置页
- 显示最新短信，顶部继续加载更早消息
- 自动识别短信中的链接和手机号
- 自动提取常见验证码并支持一键复制
- 支持系统生物识别 / 锁屏凭据解锁
- 支持调用服务端接口清空全部短信记录

### 首次配置

启动应用后，填写以下信息：

- 服务器地址
- 访问令牌（Token）
- 是否启用系统验证解锁

其中“服务器地址”支持以下形式：

- 域名，例如 `sms.example.com`
- 带项目路径的地址，例如 `example.com/my-sms`
- 完整接口地址，例如 `https://example.com/appapi/index.php`

## 权限与隐私

当前客户端只声明了以下权限：

- `INTERNET`
- `USE_BIOMETRIC`

项目没有申请读取本机短信、通讯录或通话记录权限。短信内容来自你自己的服务端接口，而不是直接从手机系统短信数据库读取。

注意事项：

- 访问令牌会保存在本地 DataStore 中
- 如果你使用局域网 / 本地 HTTP 服务，令牌会以明文网络请求方式传输
